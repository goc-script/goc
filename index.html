GOC

Set of utility functions for easy canvas game development

Installation

  • Browser
            
              <script src="http://goc.rf.gd/goc.js"> </script>

                                      (or) 

              <script src="https://gocscript.000webhostapp.com/goc.js"> </script>

Solid Game Loop

Solid Gameloop with delta time (time between two frames) and context

Async Asset Preloader

Load images, sprites before animate them on screen

Virtual JoyStick

Control any game object with joystick with just one line of code

Sprite Sheet Animation

Animate any type of sprite sheets either SINGLE IMAGE or SEPERATE IMAGES

TileMap Generator

Easy UI to generate Tile Maps with Collision Detection

Collision Detection

Detect and resolve collision between any tow or more Game Entities
